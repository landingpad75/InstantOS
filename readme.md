# InstantOS
A rewrite of the closed source graphical x86_64 operating system

# Features
- [x] Terminal Emulator
- [x] Keyboard support

# Upcoming features
- [ ] Mouse support

# Building
To build this OS, you need:

- [x86_64-elf toolchain](https://wiki.osdev.org/GCC_Cross-Compiler) (replace i686-elf with x86_64-elf)
